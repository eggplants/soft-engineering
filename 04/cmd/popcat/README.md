# `popcat`

## Help

```shellsession
$ popcat -h
usage: popcat [-f] [-t MS] [-h] [-v]

*pop* *pop* *pop*

optional arguments:
  -f, --flip            flip cat
  -t, --time <ms>       interval of frames
                        defaults to 200(ms)
  -h, --help            display this help and exit
  -v, --version         output version information and exit
```
